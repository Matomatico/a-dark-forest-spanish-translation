# 🌲 A Dark Forest – Spanish Localization (ES)

## 📌 Overview

This repository contains a Spanish (es) localization of the indie text-based RPG *A Dark Forest*, developed in Godot.

The original game did not include Spanish, so this project expands its accessibility to a wider audience.

---

## 🎯 My Contribution

* Added a full Spanish localization column (`es`) to the original CSV file
* Preserved compatibility with the existing localization system
* Adapted tone and style to match the game's **dark, pessimistic, and atmospheric narrative voice**
* Iterated translations based on **in-game testing**, not just static text

---

## 🧠 Localization Approach

This was not a literal translation. Key decisions included:

* **Tone calibration**: maintaining narrative weight and ambiguity
* **Lexical economy**: avoiding verbosity to match the original style
* **Consistency**: repeated terms and phrases were standardized
* **UI awareness**: adjusted phrasing to prevent overflow and formatting issues

---

## 🛠 Technical Notes

* Format: Godot CSV localization file
* Encoding: UTF-8 (no BOM)
* Structure preserved to ensure drop-in compatibility
